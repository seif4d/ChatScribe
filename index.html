📜 ChatScribe

الكاتب الشخصي الخاص بك لـ ChatGPT. استخرج، نسّق، وحمّل محادثاتك بسهولة كملفات Markdown جميلة باستخدام اسكربت واحد.

عرض على GitHub

كيف يعمل؟

1

افتح الكونسول

في صفحة محادثة ChatGPT، اضغط `F12` واذهب إلى تبويب Console.

2

انسخ وألصق

انسخ الكود بالكامل من الملف وألصقه في الكونسول.

3

حمّل ملفك

اضغط `Enter` وشاهد ملف Markdown الخاص بك وهو يتحمل فورًا! 🎉
